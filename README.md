## 📂 Projects

### 📚 [Library Management System](https://github.com/francoisxavierdussault/GestionBibliothequeJAVA) — Java
A console-based library management app built with a layered architecture (Model / Repository / Service / UI).

- Borrow and return books with business rule enforcement
- Per-member borrow quota validation
- Overdue loan tracking
- Custom exceptions (`LivreDejaEmprunteException`, `QuotaEmpruntDepasseException`)
- Built with Maven, follows the Repository pattern

**Tech:** Java, Maven, OOP, Repository Pattern

---

### 🗂️ [FAQ CRUD Application](https://github.com/francoisxavierdussault/csharp-crud-application) — C# / WinForms
A Windows desktop application for managing FAQ entries, structured in 3 distinct layers (UI / BLL / DAL).

- Full CRUD: add, search, filter by category, soft-delete
- Entity Framework Core with SQL Server LocalDB
- DataGridView UI with category filtering and keyword search
- Category statistics and FAQ count
- Data validation at the service layer

**Tech:** C#, WinForms, Entity Framework Core, SQL Server, 3-tier architecture
